# Frontier 2075 Brand Guidelines

## Positioning
Frontier 2075 is a hopeful space-opera interface about leaving Earth without forgetting it. Labels should feel like destinations people can imagine visiting, not engineering inventory.

## Naming Voice
- Use human, memorable place names: First Light, The Crossing, Guardian Net, Imbrium Haven.
- Pair technical scale with emotional meaning: "orbital data haven" instead of only "data center."
- Prefer short title-card phrases for hero labels: `GATEWAY // THE CROSSING`, `IMBRIUM // HAVEN`.
- Keep UI instructions plain and useful, but add a small sense of wonder where it does not hurt clarity.

## Type System
- Hero titles: Bungee Inline / Orbitron fallback, uppercase, warm gold outline, dark cinematic core, amber glow.
- Subsystem labels: Russo One / Oxanium fallback, uppercase, off-white or amber, lighter glow.
- Telemetry: Share Tech Mono, muted off-white, quiet and factual.

## Palette
- Hero gold: `#ffd23f`
- Frontier amber: `#ff9a3c`
- Off-white: `#e8e8e0`
- Charcoal core: `#090b10`
- Telemetry grey: `rgba(232, 232, 224, 0.72)`

## Reference Boundary
Evoke classic space-opera emotion through scale, outline typography, warm gold, and bold title composition. Do not reproduce the Star Wars logo, exact wordmarks, faction marks, or trademarked branding.
